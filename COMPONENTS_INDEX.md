## 🔍 使用方法

### 在VS Code中快速定位

1. **按 `Ctrl+F` (Windows) 或 `Cmd+F` (Mac)** 打开搜索
2. **输入搜索关键词**，例如：
   - `========== 5. 开始画面` - 定位到开始画面组件
   - `========== 7. 存档/读档` - 定位到存档界面
   - `MAIN APP` - 定位到主应用逻辑

### 注释格式说明

所有组件使用统一的注释格式：

```javascript
// ========== [序号]. [中文名称] [English Name] ==========
const ComponentName = () => {
    // 组件代码
};
```

## 📝 维护说明

- 添加新组件时，请按照相同格式添加注释
- 更新此索引文档以保持同步
- 废弃的组件请在注释中标注 `(已废弃/未使用)`
