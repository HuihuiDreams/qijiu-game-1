# AO3 修仙小说 Work CSS 使用说明

## 如何使用

### 1. 在AO3上应用CSS

1. 登录你的AO3账号
2. 进入你要发布或编辑的作品
3. 在作品编辑页面，找到 **"Work Skin"** 部分
4. 将 `ao3_xianxia_style.css` 的内容完整复制粘贴进去
5. 保存并预览效果

### 2. 基础功能

这个CSS会自动为你的作品添加：

- ✨ **古典卷轴风格背景** - 淡雅的宣纸质感
- 📜 **优雅的排版** - 使用宋体/明体，适合长篇阅读
- 🎋 **传统装饰元素** - 顶部和底部的云纹装饰
- 📱 **响应式设计** - 在手机上也能完美显示
- 🖨️ **打印优化** - 适合下载保存

### 3. 特殊样式类（可选）

你可以在写作时使用这些HTML标签来增强效果：

#### 章节标题
```html
<p class="chapter">第一章 · 初入仙门</p>
```

#### 修仙术语/境界
```html
<span class="cultivation-term">灵根</span>
<span class="realm">金丹期</span>
<span class="technique">御剑术</span>
<span class="spiritual-energy">灵气</span>
```

#### 内心独白/引用
```html
<blockquote>这就是传说中的仙人吗？</blockquote>
```

#### 作者注释
```html
<div class="note">
作者注：本章涉及原著第三卷剧情
</div>
```

#### 强调文本
```html
<em>重要术语</em>（会有着重号）
<strong>非常重要</strong>（加粗并有微光效果）
```

#### 分隔线
```html
<hr>
```
会显示为带装饰的古典分割线

### 4. 效果预览

- **段落**：自动首行缩进2字符，行距舒适，两端对齐
- **标题**：居中显示，带装饰符号，楷体字形
- **背景**：仿古宣纸质感，带细微纹理
- **颜色**：采用棕褐色系，护眼且有古典韵味

### 5. 自定义调整

如果你想修改样式，可以调整以下部分：

#### 改变主色调
找到这一行并修改颜色值：
```css
color: #2c2416;  /* 主文字颜色 */
color: #4a3f2f;  /* 标题颜色 */
```

#### 调整字体大小
```css
font-size: 1.1rem;  /* 正文大小 */
```

#### 修改背景颜色
```css
background: linear-gradient(to bottom, 
  #faf8f3 0%,   /* 顶部颜色 */
  #f5f1e8 50%,  /* 中间颜色 */
  #faf8f3 100%  /* 底部颜色 */
);
```

#### 改变装饰符号
找到 `content` 属性并替换：
```css
content: "◈ ◈ ◈";  /* 可以改成其他符号，如 ❀ ✿ ◆ ❖ */
```

### 6. 常见问题

**Q: 为什么我的作品显示不出效果？**
A: 确保你在编辑作品时勾选了 "Use Workskin"选项

**Q: 可以只用部分样式吗？**
A: 可以！删除CSS中你不需要的部分即可

**Q: 在深色模式下效果如何？**
A: 这个CSS主要针对浅色背景设计。如果需要深色版本，我可以另外制作

**Q: 会影响其他人阅读吗？**
A: 读者可以选择关闭Work Skin，所以不会强制所有人使用你的样式

